# Product Overview: Financial Dashboard

## Propósito del Producto
Es una aplicación web diseñada para visualizar y analizar datos financieros. Actúa como un panel de control (Dashboard) que permite a los usuarios monitorear métricas clave de rendimiento (KPIs), movimientos de dinero y datos mensuales históricos.

## Evidencia en el Código
- El frontend renderiza componentes como `DashboardHeader` y utiliza funciones como `computeKPIs` y `computeMonthlyData` para calcular métricas a partir de un flujo de movimientos financieros.
- El backend expone endpoints (API REST) para servir esta información financiera.
